Looping/Iterative Statements

for:

*for is a keyword in javascript
*for is executed until the condition is false

syntax:

for(initialization;condition;increment/decrement)
{
//Statements//
}

example: for loop

for(var i=0;i<=5;i++)
{
console.log("num:"+i)
}

0<=5=>true
num:0
0++ = 0+1
i=1

1<=5=>true
num:1
1++ = 1+1
i=2

2<=5=>true
num:2
2++ = 1+2
i=3

3<=5=>true
num:3
3++ = 1+3
i=4

4<=5=>true
num:4
4++ = 1+4
i=5

5<=5=>true
num:5
5++ = 1+5
i=6

6<=5=>false
6 is greater than 5 so loop terminates.
